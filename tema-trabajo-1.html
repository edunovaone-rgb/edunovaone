🖥️ El Computador

Un computador (o computadora) es una máquina electrónica capaz de procesar información. Se compone de dos elementos fundamentales: el hardware (partes físicas) y el software (programas e instrucciones).

🔩 Hardware — Partes físicas

  • CPU (Unidad Central de Proceso): el "cerebro", procesa datos
  • Memoria RAM: memoria temporal de trabajo
  • Disco duro (HDD/SSD): almacenamiento permanente
  • Placa madre: conecta todos los componentes
  • Monitor: dispositivo de salida visual
  • Teclado y ratón: dispositivos de entrada
  • Impresora: salida en papel

💾 Software — Tipos

  • Sistema operativo: Windows, macOS, Linux, Android
  • Aplicaciones: Word, Chrome, Photoshop, juegos
  • Utilitarios: antivirus, compresores, reproductores
  • Software libre: código abierto, gratuito (LibreOffice)
  • Software propietario: licencia de pago (Microsoft Office)
Computador = Hardware + Software + Datos

✏️ Ejercicios

  • Ejercicio 1 Clasifica los siguientes como hardware o software: monitor, Windows 11, teclado, Microsoft Word, disco duro, antivirus.
    Hardware: monitor, teclado, disco duro. Software: Windows 11, Microsoft Word, antivirus.
  • Ejercicio 2 ¿Cuál es la diferencia entre la memoria RAM y el disco duro?
    La RAM es memoria temporal: guarda datos mientras el computador está encendido y se borra al apagarse. El disco duro es almacenamiento permanente: guarda archivos aunque se apague el equipo.
  • Ejercicio 3 Menciona 3 dispositivos de entrada y 3 de salida.
    Entrada: teclado, ratón, micrófono. Salida: monitor, impresora, altavoces.

🪟 Sistema Operativo Windows

Un sistema operativo (SO) es el software base que permite al usuario interactuar con el hardware y ejecutar aplicaciones. Windows es el más usado en el mundo en computadores de escritorio.

🖥️ Elementos del escritorio

  • Escritorio: área principal de trabajo
  • Barra de tareas: muestra programas abiertos
  • Menú Inicio: acceso a programas y configuración
  • Iconos: accesos directos a archivos y programas
  • Papelera de reciclaje: archivos eliminados
  • Explorador de archivos: navegar por carpetas

📁 Gestión de archivos

  • Crear carpeta: clic derecho → Nueva carpeta
  • Copiar: Ctrl+C · Pegar: Ctrl+V · Cortar: Ctrl+X
  • Renombrar: F2 o clic derecho → Cambiar nombre
  • Eliminar: Suprimir (va a la papelera)
  • Buscar: Windows + S o barra de búsqueda
  • Extensiones: .docx, .pdf, .jpg, .mp4, .exe

✏️ Ejercicios

  • Ejercicio 1 ¿Qué combinación de teclas usas para copiar, cortar y pegar en Windows?
    Copiar: Ctrl+C · Cortar: Ctrl+X · Pegar: Ctrl+V
  • Ejercicio 2 ¿Qué tipo de archivo es un .docx? ¿Y un .jpg?
    .docx = documento de Word (texto). .jpg = imagen fotográfica comprimida.

📝 Procesador de Texto — Microsoft Word

Un procesador de texto es un programa para crear, editar y dar formato a documentos escritos. Microsoft Word y Google Docs son los más utilizados. Aprenderás las funciones esenciales para presentar trabajos escolares correctamente.

🖊️ Formato de texto

  • Negrita: Ctrl+N (resalta palabras importantes)
  • Cursiva: Ctrl+K (títulos de libros, palabras extranjeras)
  • Subrayado: Ctrl+S
  • Tamaño de fuente: recomendado 12 pts para textos
  • Fuente: Arial o Times New Roman para trabajos formales
  • Alineación: izquierda, centrado, derecha, justificado

📄 Formato de página

  • Márgenes: mínimo 2.5 cm en trabajos escolares
  • Interlineado: 1.5 o doble espacio para trabajos
  • Encabezado/pie: título, número de página, nombre
  • Guardar: Ctrl+G (guarda en .docx)
  • Guardar como PDF: Archivo → Guardar como → PDF
  • Imprimir: Ctrl+P

✏️ Ejercicios

  • Ejercicio 1 ¿Qué atajo de teclado usas para poner texto en negrita, cursiva y subrayado en Word?
    Negrita: Ctrl+N · Cursiva: Ctrl+K · Subrayado: Ctrl+S
  • Ejercicio 2 — Práctico Abre Word y crea un documento con: tu nombre en negrita y centrado, un título con tamaño 16, y un párrafo justificado de al menos 5 líneas. Guárdalo en tu escritorio.
    Ejercicio práctico. Verificar: nombre en negrita y centrado (Ctrl+N + Ctrl+T), título tamaño 16, párrafo con alineación justificada (Ctrl+J), guardado con Ctrl+G.

Internet y Seguridad

Navegadores, búsqueda segura y riesgos en línea

NavegadorBuscadorSeguridad

🌐 Internet y Seguridad Digital

Internet es una red global de computadores interconectados. Permite acceder a información, comunicarse y trabajar desde cualquier lugar. Pero también presenta riesgos que debes conocer para protegerte.

🔍 Búsqueda eficiente

  • Usa palabras clave específicas: "volcanes del Perú" no solo "volcanes"
  • Comillas para frase exacta: "cambio climático en Perú"
  • site: para buscar en un sitio: site:minedu.gob.pe matemática
  • Evalúa fuentes: .edu y .gob son más confiables
  • Verifica la fecha de publicación

🔒 Seguridad en línea

  • Contraseña fuerte: mínimo 8 caracteres, mayúsculas, números y símbolos
  • No compartir: contraseñas, dirección, número de teléfono
  • Phishing: correos falsos que roban datos — no hacer clic en links sospechosos
  • Ciberbullying: acoso en línea — reportar y bloquear
  • Privacidad: revisar configuración de privacidad en redes

✏️ Ejercicios

  • Ejercicio 1 ¿Cuáles son las características de una contraseña segura?
    Mínimo 8 caracteres, combina letras mayúsculas y minúsculas, números y símbolos (! @ # $). No uses tu nombre, fecha de nacimiento ni palabras del diccionario. Usa una distinta para cada cuenta.
  • Ejercicio 2 ¿Qué es el phishing y cómo puedes protegerte de él?
    El phishing es cuando alguien te envía un mensaje falso (email, SMS) fingiendo ser una empresa o persona conocida para robarte contraseñas o datos. Protección: no hacer clic en links sospechosos, verificar la dirección del remitente, nunca ingresar contraseñas desde links en correos.

ÉticaDerechos de autorHuella digital

🛡️ Ciudadanía Digital

Un ciudadano digital es quien usa responsablemente las tecnologías para participar en la sociedad, respetando leyes, derechos de otros y cuidando su propia reputación en línea.

📋 Principios del ciudadano digital

  • Respeto: tratar a otros con dignidad en línea
  • Responsabilidad: asumir consecuencias de lo que publicas
  • Derechos de autor: citar fuentes, no plagiar
  • Privacidad: respetar los datos de los demás
  • Veracidad: no difundir noticias falsas (fake news)

👣 Huella digital

  • Es el rastro que dejas en internet con cada acción
  • Activa: fotos, comentarios, publicaciones que tú haces
  • Pasiva: datos recopilados sin que lo sepas (cookies)
  • Una vez publicado, es difícil borrar completamente
  • Cuida lo que compartes: puede afectar tu futuro

✏️ Ejercicios

  • Ejercicio 1 ¿Qué es la huella digital activa? Da dos ejemplos de tu propia huella digital activa.
    La huella digital activa son las acciones que tú decides hacer en internet. Ejemplos personales: subir una foto a Instagram, escribir un comentario en YouTube, publicar en WhatsApp.
  • Ejercicio 2 Encuentras en Google una imagen perfecta para tu tarea. ¿Cómo la usas respetando los derechos de autor?
    Debes: 1) Buscar imágenes con licencia libre (Google → Herramientas → Derechos de uso: Creative Commons). 2) Citar la fuente: escribir debajo de la imagen el autor y la URL. 3) Nunca presentarla como tuya propia.
