🔣 Factorización Algebraica

Factorizar es expresar un polinomio como producto de factores más simples. Es la operación inversa del desarrollo algebraico y es fundamental para resolver ecuaciones.

📌 Métodos de factorización

  • Factor común: ax + ay = a(x+y) → 6x²+9x = 3x(2x+3)
  • Diferencia de cuadrados: a²-b² = (a+b)(a-b) → x²-25 = (x+5)(x-5)
  • Trinomio cuadrado perfecto: a²±2ab+b² = (a±b)² → x²+6x+9 = (x+3)²
  • Aspa simple: para ax²+bx+c; buscar factores de a·c que sumen b
  • Suma/Dif. de cubos: a³±b³ = (a±b)(a²∓ab+b²)

📝 Método del Aspa Simple

  • Factorizar: x²+5x+6
  • Buscar a, b tal que: a·b = 6 y a+b = 5
  • a=2, b=3 (2×3=6; 2+3=5) ✓
  • Resultado: (x+2)(x+3)
  • Verificar: (x+2)(x+3)=x²+5x+6 ✓
  • Ejemplo 2: x²-7x+12 → (x-3)(x-4)
a²−b² = (a+b)(a−b) | a²+2ab+b² = (a+b)² | ax²+bx+c = (px+q)(rx+s)

✏️ Ejercicios

  • Ejercicio 1 Factoriza: a) 4x²−9   b) x²+8x+16   c) 2x²+7x+3
    a) 4x²−9 = (2x)²−3² = (2x+3)(2x−3) · b) x²+8x+16 = (x+4)² = (x+4)² · c) 2x²+7x+3: buscar factores → (2x+1)(x+3) → verificar: 2x²+6x+x+3 = 2x²+7x+3 ✓
  • Ejercicio 2 Factoriza completamente: 3x³−12x
    Primero saco factor común: 3x³−12x = 3x(x²−4) = 3x·(x+2)(x−2) → Respuesta: 3x(x+2)(x−2)
  • Ejercicio 3 Factoriza: x²−x−12
    Busco a·b = −12 y a+b = −1 → a=−4, b=3 (−4×3=−12; −4+3=−1) → (x−4)(x+3)

⚖️ Inecuaciones Lineales

Una inecuación es una desigualdad que expresa la relación entre dos expresiones algebraicas. Se resuelve igual que una ecuación, con una regla especial: al multiplicar o dividir por un número negativo, se invierte el signo de desigualdad.

📌 Reglas de inecuaciones

  • Se puede sumar o restar igual a ambos lados
  • Se puede multiplicar/dividir por positivo sin cambiar el signo
  • Si multiplicas/divides por negativo: cambia el signo
  • Ejemplo: −2x < 6 → x > −3 (se invirtió < por >)
  • Solución: un intervalo, se representa en recta numérica

📝 Sistemas de inecuaciones

  • Sistema: resolver cada inecuación y encontrar la intersección
  • Ej: x+2 > 5 → x > 3 · 2x−1 ≤ 9 → x ≤ 5
  • Solución: 3 < x ≤ 5 (intervalo: (3, 5])
  • Notación de intervalos: [a,b], (a,b), [a,b), (a,b]
  • [: incluye el extremo | (: no incluye el extremo
Si c < 0: ax < b → x > b/a (invertir signo)

✏️ Ejercicios

  • Ejercicio 1 Resuelve: 3x−5 > 7
    3x−5 > 7 → 3x > 12 → x > 4. Solución: x > 4 ó intervalo (4, +∞)
  • Ejercicio 2 Resuelve el sistema: 2x+1 ≥ 5 y x−3 < 2
    Primera: 2x+1≥5 → 2x≥4 → x≥2. Segunda: x−3<2 → x<5. Intersección: 2 ≤ x < 5, intervalo [2, 5)

📍 Geometría Analítica

La geometría analítica estudia figuras geométricas mediante coordenadas en el plano cartesiano. Las fórmulas de distancia, punto medio y ecuación de la recta son las herramientas fundamentales.

📐 Fórmulas esenciales

  • Distancia: d = √[(x₂−x₁)²+(y₂−y₁)²]
  • Punto medio: M = ((x₁+x₂)/2 , (y₁+y₂)/2)
  • Pendiente: m = (y₂−y₁)/(x₂−x₁)
  • Ec. de la recta (punto-pendiente): y−y₁ = m(x−x₁)
  • Forma pendiente-intersección: y = mx+b
  • Forma general: Ax+By+C = 0

📝 Ejemplo resuelto

  • A(1,2) y B(4,6)
  • d = √[(4−1)²+(6−2)²] = √[9+16] = √25 = 5
  • M = ((1+4)/2, (2+6)/2) = (2.5, 4)
  • m = (6−2)/(4−1) = 4/3
  • Ecuación: y−2 = (4/3)(x−1) → y = (4/3)x + 2/3
d = √[(x₂−x₁)²+(y₂−y₁)²] | M = ((x₁+x₂)/2, (y₁+y₂)/2)

✏️ Ejercicios

  • Ejercicio 1 Halla la distancia y el punto medio entre P(−2, 3) y Q(4, −5).
    d = √[(4−(−2))²+(−5−3)²] = √[36+64] = √100 = 10. M = ((−2+4)/2, (3+(−5))/2) = (1, −1) → M = (1, −1)
  • Ejercicio 2 Halla la ecuación de la recta que pasa por (2, 1) y (5, 7).
    m = (7−1)/(5−2) = 6/3 = 2. y−1 = 2(x−2) → y−1 = 2x−4 → y = 2x − 3

Probabilidades

Espacio muestral, eventos y regla de Laplace

P(A)LaplaceComplemento

🎲 Probabilidades

La probabilidad mide la posibilidad de que ocurra un evento. Se basa en el concepto de experimento aleatorio: un proceso cuyo resultado no puede predecirse con certeza.

📌 Conceptos clave

  • Experimento aleatorio: lanzar un dado, sacar una carta
  • Espacio muestral (S): conjunto de todos los resultados posibles
  • Evento (A): subconjunto del espacio muestral
  • Regla de Laplace: P(A) = casos favorables / casos totales
  • 0 ≤ P(A) ≤ 1 siempre
  • P(evento seguro) = 1, P(evento imposible) = 0

📝 Propiedades

  • Complemento: P(A') = 1 − P(A)
  • Unión: P(A∪B) = P(A)+P(B)−P(A∩B)
  • Independientes: P(A∩B) = P(A)·P(B)
  • Ejemplo: dado → S = {1,2,3,4,5,6}
  • P(par) = 3/6 = 1/2
  • P(mayor que 4) = 2/6 = 1/3
P(A) = n(A)/n(S) | P(A) + P(A') = 1

✏️ Ejercicios

  • Ejercicio 1 Se lanza un dado. ¿Cuál es la probabilidad de obtener un número primo?
    S = {1,2,3,4,5,6}. Primos: {2,3,5} → 3 casos favorables. P(primo) = 3/6 = 1/2 = 0.5 = 50%
  • Ejercicio 2 En una bolsa hay 4 bolas rojas, 3 azules y 5 verdes. ¿Cuál es la P(no roja)?
    Total = 12. P(roja) = 4/12 = 1/3. P(no roja) = 1 − 1/3 = 2/3 ≈ 0.667 (azul o verde: 8/12 = 2/3 ✓)

Trigonometría

Ley de senos, cosenos e identidades

Ley senosLey cosenosIdentidades

📐 Trigonometría

Las leyes de senos y cosenos permiten resolver triángulos no rectángulos. Las identidades trigonométricas son igualdades que se cumplen para cualquier ángulo.

📌 Leyes trigonométricas

  • Ley de senos: a/sen(A) = b/sen(B) = c/sen(C)
  • Úsala cuando conoces: AAS, ASA, o SSA
  • Ley de cosenos: a² = b²+c²−2bc·cos(A)
  • Úsala cuando conoces: SSS o SAS
  • Si A=90°: ley de cosenos → Pitágoras (cos90°=0)

📝 Identidades trigonométricas

  • Pitagórica: sen²(x)+cos²(x)=1
  • 1+tan²(x) = sec²(x)
  • 1+cot²(x) = csc²(x)
  • Ángulo doble: sen(2x)=2sen(x)cos(x)
  • cos(2x)=cos²(x)−sen²(x)
a/sen A = b/sen B = c/sen C | a² = b²+c²−2bc·cos A

✏️ Ejercicios

  • Ejercicio 1 En un triángulo: A=30°, B=45°, a=10. Halla b usando la ley de senos.
    a/sen(A) = b/sen(B) → 10/sen(30°) = b/sen(45°) → 10/0.5 = b/(√2/2) → 20 = b/(0.707) → b = 20×0.707 = b ≈ 14.14
  • Ejercicio 2 Verifica la identidad: sen²(x)+cos²(x)=1 para x=60°.
    sen(60°)=√3/2 → sen²(60°)=3/4. cos(60°)=1/2 → cos²(60°)=1/4. Suma: 3/4+1/4 = 4/4 = 1 ✓

Estadística Inferencial

Muestras, distribución normal e interpretación

MuestraDistribuciónNormal

📊 Estadística Inferencial Básica

La estadística inferencial usa datos de una muestra para hacer conclusiones sobre una población. La distribución normal es fundamental: es simétrica, con forma de campana.

📌 Conceptos clave

  • Población: conjunto total de individuos u objetos
  • Muestra: subconjunto representativo de la población
  • Distribución Normal: simétrica, media=mediana=moda
  • 68% de datos: entre μ−σ y μ+σ
  • 95% de datos: entre μ−2σ y μ+2σ
  • 99.7% de datos: entre μ−3σ y μ+3σ

📝 Interpretación de gráficas

  • Histograma: barras que representan frecuencias
  • Polígono de frecuencias: línea que une los puntos medios
  • Ojiva: frecuencias acumuladas
  • Diagrama de caja (boxplot): muestra Q1, mediana, Q3, extremos
  • La forma de la distribución indica si hay sesgo
Distribución Normal: μ±σ (68%) | μ±2σ (95%) | μ±3σ (99.7%)

✏️ Ejercicios

  • Ejercicio 1 Las notas de un examen siguen distribución normal con media=14 y desviación estándar=2. ¿Qué porcentaje de alumnos tiene nota entre 12 y 16?
    μ=14, σ=2. El intervalo [12, 16] = [μ−σ, μ+σ]. Por la regla empírica, entre μ−σ y μ+σ se encuentra el 68% de los datos.
  • Ejercicio 2 ¿Cuál es la diferencia entre población y muestra? Da un ejemplo.
    Población: todos los individuos del grupo de interés. Muestra: un subconjunto elegido para estudiar. Ejemplo: si queremos saber las notas de todos los alumnos del Perú (población), encuestamos 5000 alumnos de diversas escuelas (muestra) y usamos esos datos para inferir sobre todos.
